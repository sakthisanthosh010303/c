# Left Right Odd Count

## Description

Given an array of numbers. Print a number in the array if the number of odd numbers to the left and the number of odd numbers to the right are equal. Else print -1.

## Boundary Conditions

1 <= array_size <= 10^5

## Test Case

### Input

1 3 4 8 5 7

### Output

-1 -1 4 8 -1 -1
